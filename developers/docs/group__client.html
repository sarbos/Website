Detailed Description

The client module provides the tools necessary to connect to and interact with the Ilium VR runtime service. This is generally where a beginner to this API will start their work. An example of very basic usage would be the following:

#include <string.h>
#include <ivr.h>
#include <ivr/runtime.h>
#include <ivr/runtimeclient.h>
ivr_client* client;
void init()
{
ivr_client_params params;
params.mode = IVR_RUNTIME_MODE_GAME;
params.options = IVR_RUNTIME_OPT_DEFAULT;
params.timeout = 2000;
ivrClientInit(&client, &params);
//Just use the first device no matter what
char* dev_names;
ivrClientListHidDevices(client, &dev_names);
char* newline = strchr(dev_names, '\n');
if (newline) newline = '\0';
ivrClientSelectHidDevice(client, &dev_names);
ivrClientListSerialDevices(client, &dev_names);
newline = strchr(params.sdev_name, '\n');
if (newline) newline = '\0';
ivrClientSelectSerialDevice(client, &dev_names);
ivrClientStart(client);
}
void update(my_game_object* obj)
{
ivr_game_packet packet;
ivrClientGetGamePacket(client, &packet);
obj->rotation = packet.imu;
if (packet.trig < 100)
myGameFireWeapon(obj);
}
void exit()
{
ivrClientDestroy(&client);
}

This example omits error checking for simplicity and readability.

Data Structure Documentation

struct ivr_client_params

Parameters for generating a runtime client.

See also
ivrRuntimeClientInit()
Data Fields
ivr_runtime_mode mode The way the runtime will operate and provide data.
ivr_runtime_options options A list of options to start a runtime with. Only for internal use currently.
uint32_t timeout The amount of time, in milliseconds, to wait for any action from the client before automatically terminating the runtime.

If you don't want a timeout, use IVR_RUNTIME_TIMEOUT_INFINITE.

struct ivr_client

A client that can connect to and communicate with an instance of the Ilium VR runtime. Access only as an opaque pointer, as the contents of the struct will change between operating systems.

)

Initializes a new ivr_client with the settings provided in ivr_client_params.

Parameters
[out]clientA pointer to a variable that will be allocated over and filled with an initialized ivr_client.
[in]paramsThe desired settings for a new instance of the runtime.
Returns
If this function is successful, it will return IVR_E_OK. Otherwise, client will be set to NULL and this function will return one of the following errors:
Value Description
IVR_E_ARG_NULL One of the arguments is null
IVR_E_PIPE_CREATE Runtime service is unavailable, or there is a permissions error.
IVR_E_OS Generic operating system error. Call GetLastError() on Windows for more detail.
