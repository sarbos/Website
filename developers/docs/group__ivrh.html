Detailed Description

All the settings provided by the ivr.h header file. All versions follow semantic versioning (http://semver.org/)

Macro Definition Documentation

#define IVR_IS_DYNAMIC

libiliumvr supports being built as both a static and dynamic library. If you are trying to dynamically link your project to libiliumvr, make sure that you define this constant before your include any part of libiliumvr or you will get a lot of linker errors.

#define <string.h>
#define <stdint.h>
#define IVR_IS_DYNAMIC
#include <ivr.h>
#include <ivr/client.h>

Gets the version of libiliumvr being called.

If you are statically linking, prefer to use the IVR_LIB_VERSION_* costants. If you are dynamically linking, this is the only way to reliably get the version.
