Detailed Description

The math module provides linear algebra structs and functions to assist with transformations and sensor data. As a client, you will most likely only ever use the structs to extract coordinates.

struct ivr_quat

A quaternion, representing orientations without gimbal lock.

Stored as <w, x, y, z>. Be careful of systems that store it as <x, y, z, w>.

Data Fields
float w The W element of the quaternion.
float x The X element of the quaternion.
float y The Y element of the quaternion.
float z The Z element of the quaternion.
